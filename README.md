## 前言

其实vue的文档要写出和官网不一样的感觉很难 应为官方文档写的已经很好了  
但目前很少人去读完vue的整个文档  
目前现在我看到不少人 说webpack 不用学 都有人配好了  
入门开始 直接 vuecli 这样其实是不对的  
我准备 以vue源码为基础讲vue的技术
我们的目标是培养看懂源码的vue工程师
